Shared Grocery List

AI-Powered Grocery Sharing for Modern Households

Mikhail Petrunin m.petrunin@innopolis.university Group DSAI-04

Context & Problem

👥

End Users

Roommates, couples, and families who share grocery shopping responsibilities and need to coordinate household purchases.

😤

The Problem

No shared visibility into household supplies — someone forgot to buy milk, someone bought bread twice. Leads to waste, frustration, and duplicated purchases.

💡 Solution: A simple shared grocery list accessible from any device — create a room, share the code with roommates, and everyone can add items manually or via natural language AI input.

Implementation

⚛️ React
🐍 FastAPI
🐘 PostgreSQL
🐳 Docker
🤖 AI Agent

Version 1

  • Basic grocery list (single user)
  • Add/remove items
  • Mark items as "bought"
  • Simple CRUD interface
  • SQLite database

Version 2

  • Shared rooms with 6-char codes
  • Multi-user real-time sync
  • AI natural language input
  • Room isolation & security
  • PostgreSQL + Docker deployment
